# Security Policy

## Supported versions

This repository tracks the PHP 8.4+ modernization of Question2Answer. Security
fixes are provided for the active development line only. The legacy 1.8.x series
is not maintained here.

## Reporting a vulnerability

Please report security issues **privately** — do not open a public issue or pull
request for them.

- Preferred: use GitHub's **"Report a vulnerability"** (Security Advisories) on
  this repository.
- We aim to acknowledge reports within a few days and to keep you informed about
  the fix and disclosure timeline.

Please include as much detail as possible: affected version/commit, reproduction
steps, impact, and any proof of concept. Responsible disclosure is appreciated —
we will credit reporters who wish to be named once a fix is released.
